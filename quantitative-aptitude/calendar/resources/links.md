# Calendar — Resources

Curated external links for further reading, extra practice, and video explanations on Calendar
aptitude problems (odd days, leap years, day-of-week calculations).

## 📖 Articles & Theory

- **GeeksforGeeks — Tricks to Solve Calendar Problems**
  Covers the month-code / year-code shortcut method in detail with worked examples.
  https://www.geeksforgeeks.org/maths/tricks-to-solve-calendar-problems/

- **GeeksforGeeks — Calendar Questions (Aptitude and Reasoning)**
  Solved examples ranging from basic day-lookup to leap-year edge cases.
  https://www.geeksforgeeks.org/aptitude/calendar-questions-aptitude-and-reasoning/

- **IndiaBIX — Calendar Aptitude Questions**
  Large practice bank with step-by-step odd-days solutions; also offers downloadable PDFs.
  https://www.indiabix.com/aptitude/calendar/

- **PrepInsta — Calendar Tips, Tricks & Shortcuts**
  Placement-focused shortcuts, formatted specifically for TCS/Infosys/Wipro-style tests.
  https://prepinsta.com/calendar/tips-and-tricks-and-shortcuts/

- **PrepInsta — Calendar Questions and Answers**
  Additional practice set with formulas summarized up front.
  https://prepinsta.com/calendar/questions/

- **Talent Battle — Calendar Latest Aptitude Questions, Tips & Tricks**
  More solved examples, useful for cross-checking your odd-days calculations.
  https://talentbattle.in/practice-quantitative-aptitude-questions/calendar

- **Testbook — Clock and Calendar MCQs (with Data Sufficiency variants)**
  Good for the "data sufficiency" question style seen in bank/government exams.
  https://testbook.com/objective-questions/mcq-on-clock-and-calendar--5eea6a0e39140f30f369e4b6

## 🧭 How to Use These Resources

1. Start with this repo's own `notes/` files for the core method — they're self-contained.
2. Use IndiaBIX / PrepInsta / Talent Battle for **volume practice** once the method is clear.
3. Watch one of the YouTube videos above if you prefer a spoken walkthrough of the odd-days method
   or the month-code shortcut.
4. Use Testbook's data-sufficiency set to practice the "CAT-style" reasoning variant of calendar
   questions, which is different from direct day-lookup questions.
