# 🧠 Aptitude Grind

A curated collection of notes, practice questions, and resource links for aptitude test preparation — covering Quantitative Aptitude, Logical Reasoning, Verbal Ability, and Data Interpretation.

Built for CSE placement tests, CAT, and other similar competitive/aptitude exams, this repo aims to be a one-stop reference with concise theory, solved examples, practice questions, and links to the best external resources.

## 📦 What's Inside

- **Notes** — Short, exam-focused theory and formula sheets for quick revision.
- **Questions** — Practice problems organized by topic and difficulty, with solutions/explanations.
- **Resources** — Hand-picked links to articles, videos, PDFs, and websites for deeper learning.

## 🎯 Who This Is For

This repo is aimed at aptitude tests commonly encountered in:

- **CSE Placement Drives** — on-campus and off-campus technical/non-technical hiring tests
- **CAT** and other MBA entrance exams
- Other competitive/aptitude exams that test similar Quantitative, Logical Reasoning, Verbal Ability, and Data Interpretation skills

---

## 🚀 How to Use

1. Pick a topic area (`quantitative-aptitude/`, `logical-reasoning/`, `verbal-ability/`, or `data-interpretation/`).
2. Go into the relevant subtopic folder.
3. Inside each subtopic:
   - `notes/` — concise theory and formula sheets for quick revision
   - `questions/` — practice problems with solutions/explanations
   - `resources/` — curated external links (articles, videos, PDFs) for deeper learning
4. Track your progress by starring topics you've completed, or fork the repo and maintain your own checklist.

## 🤝 Contributing

Contributions are welcome! If you'd like to add notes, questions, or resource links:

1. Fork this repository.
2. Create a new branch (`git checkout -b add-topic-name`).
3. Add your content under the correct topic → subtopic → `notes/` / `questions/` / `resources/` folder, following the existing format.
4. Commit your changes and open a Pull Request.

Please keep notes concise, cite sources where relevant, and double-check solutions before submitting practice questions.
